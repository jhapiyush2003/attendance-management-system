# Attendance Management System

Python desktop application for managing student attendance using SQLite database and summary reports.

## Features

- Add new student records
- Mark daily attendance
- View attendance history
- Update and delete records
- SQLite database storage
- Generate attendance summary reports
- Easy-to-use desktop interface

## Technologies Used

- Python
- Tkinter
- SQLite3

## Project Goal

To simplify student attendance tracking and record management using a desktop-based system.
